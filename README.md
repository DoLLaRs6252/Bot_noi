 # วิธีติดตั้ง

เลือก file path ที่ต้องการจะเก็บไฟล์ไว้
```
cd filepath
```

clone repo
``` 
git clone https://github.com/DoLLaRs6252/Bot_noi.git
```

เปลี่ยน directory ไปที่ Bot_noi
```
cd Bot_noi
```

ติดตั้ง dependency
```
npm i
```
start server
```
npm run dev
```
